## 📘 About this repository

This repository contains the full set of **lecture notes, labs, and assignments** used in EPS 164.  

The materials are designed to accompany *Introduction to Seismology (Shearer, 3rd ed.)* and emphasize:
- hands-on analysis of real seismic data  
- computational workflows in Python  
- connections between theory and observation  

Rather than duplicating content here, the full course structure, modules, and navigation are maintained in the JupyterBook.

---

## 🧭 How to use this repo

- **Students:** Start with the course site (link above) and follow modules in order  
- **Instructors:** Use or adapt notebooks and lectures for your own courses  
- **Researchers:** Reuse workflows for seismic data analysis and teaching  

Most notebooks can be run:
- locally (via conda/pixi environment), or  
- directly in **Google Colab** (badges included in notebooks)
